# OpenZoom Commercial Offerings

OpenZoom is free software under GPL-3.0-only (see [`LICENSE`](LICENSE)), built
so that legally blind students can watch lectures through a camera magnifier
without paying for the privilege. That does not change. This document
describes what *can* be paid for, on top of the free project — never instead
of it.

## What is always free

Everything. The complete source code, every feature, every release. Anyone
may use, study, modify, and redistribute OpenZoom under the GPL, forever.
Nothing in this document gates any capability behind payment.

## What you can buy

### 1. Support contracts and SLAs

For universities, schools, disability-services offices, and assistive-
technology departments that deploy OpenZoom for their students and need
someone answerable:

- guaranteed response times for incidents,
- installation and deployment assistance (hardware selection, GPU runtime
  setup, campus imaging),
- compatibility testing for your specific camera/laptop fleet,
- a named human who owes you an answer.

### 2. Sponsored development

If you need a feature, an integration, or a "huge update" on a timeline,
you can fund its development. Sponsored work is delivered as normal GPL
code in the public repository — you are paying for priority and direction,
and every other user benefits from the result. This is the preferred way
for institutions to shape the project.

### 3. Commercial licensing

OpenZoom's copyright is held by a single owner, and contributions are
accepted under a CLA ([`CLA.md`](CLA.md)) that preserves the ability to
dual-license. If you want to incorporate OpenZoom code into a proprietary
product without GPL obligations — for example, an assistive-technology
vendor embedding the stabilization or text-clarity pipeline — a separate
commercial license can be arranged. Pricing is by inquiry.

Commercial builds may also bundle NVIDIA runtime components that the GPL
build deliberately does not ship (see the NVIDIA linking exception in
[`LICENSE`](LICENSE) and [`docs/THIRD_PARTY_LICENSES.md`](docs/THIRD_PARTY_LICENSES.md)).

### 4. Prebuilt convenience

Signed installers, managed updates, and preconfigured bundles may be
offered for a fee. The same software remains buildable from source by
anyone, free, always.

## The promise

Commercial arrangements are additive. No feature is removed from, delayed
in, or degraded in the GPL version to make a paid offering more
attractive. Accessibility features are never paywalled. If the project
name changes or the project gains maintainers, this promise travels with
it.

## How to inquire

Open a GitHub issue titled "Commercial inquiry" (no confidential details),
or contact the repository owner directly via the contact information on
the GitHub profile. Please include your organization, the offering you are
interested in (support / sponsored work / licensing), and rough scope.
